# Тестовое задание
Приложение для создания задач

## Установка и запуск

Перед запуском создайте в корне /server и /client файлы .env

Примеры можете посмотреть в .env.example

## Скрипт для засидки

Скрипт для засидки базы данных запускается с /server командой

```npm run db:seed```

## Скрипт для очистки БД

Скрипт для для очистки БД запускается с /server командой

```npm run db:destroy```

## Запуск

После успешной установки зависимостей и засидки БД - запустите приложение

### Запуск сервера осуществляется с папки /server командой 
```npm run start:dev```

### Запуск фронта осуществляется с папки /client командой 
```npm run dev```
